 🚗 Smart Accident Forecasting System

 📌 Description
A web-based application developed to analyze accident data and identify high-risk areas. The system helps in understanding accident patterns and improving road safety.

 🚀 Features
- Accident Data Management
- View Accident Reports
- Identify High-Risk Areas
- Simple Dashboard Interface

 🛠 Tech Stack
- Python
- Django
- HTML
- CSS
- SQLite

 📂 Project Structure
- manage.py → Main entry point  
- accident/ → Accident data handling  
- Roadrisk/ → Project configuration  
- reports/ → Reports data  
- media/ → Uploaded files  

 💡 What I Learned
- Django backend development  
- Database handling and CRUD operations  
- Working with real-time project structure  
- Basic data analysis
